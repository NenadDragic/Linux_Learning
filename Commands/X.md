# X

## xz – Compress and decompress files

```bash
xz filnavn.txt
```

### Top 5 mest brugte options

### 1. Komprimer fil

```bash
xz filnavn.txt
```

### 2. Dekomprimer fil

```bash
xz -d filnavn.txt.xz
```

### 3. Bevar original fil ved komprimering

```bash
xz -k filnavn.txt
```

### 4. Vis fremgang

```bash
xz -v filnavn.txt
```

### 5. Sæt komprimeringsniveau (0-9)

```bash
xz -9 filnavn.txt
```
