# W

## wc

```bash
wc -l
```

### Top 5 mest brugte options

### 1. Tæl antal linjer i en fil

```bash
wc -l filnavn.txt
```

### 2. Tæl antal ord i en fil

```bash
wc -w filnavn.txt
```

### 3. Tæl antal tegn i en fil

```bash
wc -c filnavn.txt
```

### 4. Tæl linjer fra output af en anden kommando

```bash
ls | wc -l
```

### 5. Tæl linjer i flere filer på én gang

```bash
wc -l *.txt
```
