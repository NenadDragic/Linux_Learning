# A

## alias – Create an alias for Linux commands

```bash
alias name='command'
```

### Top 5 mest brugte options

### 1. Opret en simpel alias

```bash
alias ll='ls -la'
```

### 2. Naviger hurtigt med genvej

```bash
alias ..='cd ..'
```

### 3. Farvet grep output

```bash
alias grep='grep --color=auto'
```

### 4. List alle aktive aliases

```bash
alias -p
```

### 5. Fjern en alias

```bash
unalias ll
```
