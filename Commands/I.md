# I

## id – Print user and group IDs

```bash
id
```

### Top 5 mest brugte options

### 1. Vis info for en specifik bruger

```bash
id brugernavn
```

### 2. Vis kun bruger-ID (UID)

```bash
id -u
```

### 3. Vis kun gruppe-ID (GID)

```bash
id -g
```

### 4. Vis alle gruppe-ID'er

```bash
id -G
```

### 5. Vis kun navn (ikke nummer)

```bash
id -un
```
