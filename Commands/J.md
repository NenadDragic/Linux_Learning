# J

## jobs – List active jobs

```bash
jobs
```

### Top 5 mest brugte options

### 1. Vis alle job med PID

```bash
jobs -l
```

### 2. Vis kun kørende job

```bash
jobs -r
```

### 3. Vis kun stoppede job

```bash
jobs -s
```

### 4. Send job til baggrunden

```bash
bg %1
```

### 5. Bring job til forgrunden

```bash
fg %1
```
