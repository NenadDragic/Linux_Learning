# O

## open – Open a file in its default application

```bash
open filnavn
```

### Top 5 mest brugte options

### 1. Åbn en mappe i filhåndterer

```bash
xdg-open mappe/
```

### 2. Åbn URL i browser

```bash
xdg-open https://example.com
```

### 3. Åbn fil med bestemt program

```bash
xdg-open filnavn.pdf
```

### 4. Åbn nuværende mappe

```bash
xdg-open .
```

### 5. Åbn fil (macOS)

```bash
open -a "Applikation" filnavn
```
