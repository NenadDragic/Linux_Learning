# V

## vd – Interaktiv datavisning i terminalen (VisiData)

```bash
vd filnavn.csv
```

### vd – Top 5 mest brugte options

#### 1. Åbn en CSV-fil

```bash
vd data.csv
```

#### 2. Åbn en JSON-fil

```bash
vd data.json
```

#### 3. Åbn en TSV-fil

```bash
vd data.tsv
```

#### 4. Åbn fil og vis statistik (inde i vd)

```bash
vd data.csv
# Tryk Shift+I for kolonne-statistik
```

#### 5. Gem ændringer til ny fil (inde i vd)

```bash
vd data.csv
# Tryk Ctrl+S og angiv filnavn
```

## vnstat – Netværkstrafikstatistik

```bash
vnstat
```

### vnstat – Top 5 mest brugte options

#### 1. Vis statistik for alle grænseflader

```bash
vnstat
```

#### 2. Vis statistik for bestemt grænseflade

```bash
vnstat -i eth0
```

#### 3. Vis daglig statistik

```bash
vnstat -d
```

#### 4. Vis månedlig statistik

```bash
vnstat -m
```

#### 5. Vis live trafik

```bash
vnstat -l -i eth0
```

## vuescan – Kør VueScan scanner-software

```bash
vuescan
```

### vuescan – Top 5 mest brugte options

#### 1. Start VueScan med grafisk brugerflade

```bash
vuescan
```

#### 2. Kør i kommandolinje-tilstand

```bash
vuescan -x
```

#### 3. Scan og gem direkte til fil

```bash
vuescan -x -o output.pdf
```

#### 4. Vis tilgængelige scannere

```bash
vuescan -x --devices
```

#### 5. Vis hjælp og options

```bash
vuescan --help
```
